## Quando Usar

Esta skill cobre:
- **Data Engineering**: Pipelines, ETL, modelagem de dados
- **Arquitetura**: Design de sistemas, padrões, escalabilidade
- **Backend**: APIs, microserviços, bancos de dados
- **Frontend**: React, Vue, Angular, performance
- **DevOps**: CI/CD, containers, cloud
- **Segurança**: Threat detection, incident response, red team
- **AI/ML**: Modelos, integração, MLOps

## Capacidades

### Data Engineering
- Design de pipelines de dados
- Modelagem de bancos de dados (SQL/NoSQL)
- ETL/ELT patterns
- Data warehouses e lakehouses

### Arquitetura
- Design de sistemas distribuídos
- Microserviços vs monolitos
- Patterns: CQRS, Event Sourcing, Saga
- Escalabilidade e resiliência

### Backend
- API design (REST, GraphQL)
- NestJS, Django, FastAPI, Express
- Autenticação e autorização
- Otimização de queries

### Frontend
- React, Vue, Angular, Svelte
- Performance (Core Web Vitals)
- Acessibilidade (WCAG 2.2)
- State management

### DevOps & Cloud
- Docker, Kubernetes
- CI/CD (GitHub Actions, GitLab CI)
- AWS, GCP, Azure
- Terraform, Helm

### Segurança
- Adversarial review
- AI security
- Cloud security
- Incident response
- Red team exercises
- Threat detection

### AI/ML
- Integração de modelos
- RAG (Retrieval-Augmented Generation)
- MLOps
- Fine-tuning
